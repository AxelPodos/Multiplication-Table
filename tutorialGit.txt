1 - Creacion de archivo .gitIgnore: archivos y directorios que se ignorarán en git

2 - En este caso, ignoramos los productos de la app en salida, pero creamos un .md comentando que tiene la carpeta

3 - En general, lo que se ignora son los node_modules. No tiene sentido subirlos a GIT

4 - git init: comando para inicializar 

5 - git add . : preparar archivos

6 - git commit -m : tomar la fotografia con su descripcion

7 - Creamos el repo en gitHub donde se guardará el proyecto

8 - Comandos para subir proyecto remoto
     - git remote add origin git@github.com:AxelPodos/TABLA-NODE.git
     - git branch -M main
     - git push -u origin main
